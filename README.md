# ShakeFizz

スマホを炭酸飲料の缶に見立てて振る、超楽しいエンタメゲームアプリ！  
思いっきり振って、どれだけ高く噴き出すかを競おう！ 🚀

---

## コンセプト

スマホ画面全体を「缶の中身の窓」にして、液体が激しく揺れ（sloshing）、泡が弾け、振りが強くなると炭酸が上へ噴き上がる。  
噴射の高さ（メートル単位）で友達とスコアを競う、中毒性抜群の遊びアプリです。

**デザインの方向性**: 若者向けの**エネルギッシュで没入感のある**モバイルアプリ。暗い背景 × ネオンカラー（ティール・シアン・マゼンタ・イエロー）で、モダンでクールな印象を演出。

**開発アプローチ**: UI優先開発を採用。ビジュアルデザインとユーザーインターフェースを先に構築し、段階的に機能を実装。詳細な技術仕様・データ設計は [DESIGN.md](docs/DESIGN.md) と [data-design.json](docs/data-design.json) を参照。

## 画面構成（5画面）

| 画面 | 役割 |
|------|------|
| **1. 飲み物選択** | コーラ・レモンソーダなどカラフルな缶を2×2グリッドで選択。「SELECT YOUR ULTIMATE FIZZ」で戦略性を匂わせる。画面上部に **YOUR BEST**（自己ベスト記録）を表示してモチベーション向上。**「START SHAKING」**で開始。 |
| **2. 安全警告** | プレイ前の必須ステップ。黄×黒ストライプと「SAFETY FIRST!」「HOLD TIGHT WITH BOTH HANDS!」で両手保持を促す。持ち方イラスト・免責事項あり。**「GOT IT!」**でのみ次へ（スキップ不可）。 |
| **3. プレイ** | 缶内部ビューで没入感を提供。3秒のカウントダウン後、15秒間のタイムアタック開始！画面全体が飲み物の色で満たされ、**振りの強さに応じて泡が大量発生**。液体が揺れ、泡のパーティクルが視覚的フィードバック。**数値表示なし**で、結果への期待感を最大化。時間切れで「TIME UP!」、**「VIEW RESULTS」**ボタンで結果画面へ。 |
| **4. 結果** | **RANK S〜C** バッジ、達成スコア（m）、NEW PERSONAL RECORD、TOP SPEED / TOTAL SHAKES。**SHARE YOUR SPRAY**（X・Instagram・共有）、**TRY AGAIN** / **CHANGE DRINK**。 |
| **5. グローバルランキング** | **GLOBAL ELITE** リーダーボード。GLOBAL / FRIENDS 切り替え、シーズン終了カウントダウン、トップ3（大きく表示）、4位以降のリスト、最下部に自分の順位バナー（YOU・スコア・RANK）。 |

**画面遷移**: `飲み物選択 → 安全警告 → プレイ → 結果`。各画面からランキング・飲み物選択などへ遷移可能。

### グループ対戦モード（追加予定）

人数を入力してルームを作り、同じグループ内でそれぞれゲームを行い、終了後に**グループ内の順位**を出すモード。

| ステップ | 内容 |
|----------|------|
| ルーム作成 | 参加人数（2〜8人）を入力 → **CREATE ROOM** → ルームコード or QR で共有 |
| 参加 | コード入力 or QR スキャンで **JOIN** |
| 待機 | 「Waiting for players...」「3 / 5 joined」で参加者を待つ。揃ったら **START** |
| 各自プレイ | 各人が自分の端末で通常フロー（飲み物選択 → 安全警告 → プレイ → 結果）を実施し、スコアをルームに送信 |
| グループ結果 | 全員のプレイが終わったら **GROUP RESULTS** — 1位〜のグループ内ランキングを表示 |

詳細な画面・UI仕様は **[DESIGN.md](docs/DESIGN.md#6-グループ対戦モードgroup-battle)** を参照。

## 遊び方

1. 好きな飲み物を選ぶ（ULTRA COLA / LIME BURST / BEAST FUEL など。一部はランクでアンロック）
2. **スマホを両手でガッチリ持って！** ⚠️ 安全警告を読んで「GOT IT!」
3. 3秒のカウントダウン後、15秒間思いっきり振る！（強さ・連続性で噴射高さが変わる）
4. 制限時間終了後、**「結果を見る」** ボタンを押して噴射結果を確認
5. **最高噴射高さ（m）とランク（S〜C）を競う** → 結果をシェア or ランキングで競争

## 主な機能

- **没入型プレイ**: 缶内部ビュー、液体の揺れ（Accelerometer 連動）、**泡のパーティクルで視覚的フィードバック**。振るほど泡が増える直感的な体験。数値表示なしで結果への期待感を最大化
- **自己ベスト表示**: 飲み物選択画面に自分の最高記録を表示。モチベーション向上とリプレイ意欲を刺激
- **ランクシステム**: C / B / A / **S** で評価。結果画面で RANK バッジ・NEW PERSONAL RECORD 表示
- **パフォーマンス統計**: 結果画面で TOP SPEED（km/h）、TOTAL SHAKES を表示
- **安全重視**: 振る前の視認性の高い警告画面（スキップ不可）
- **シェア**: 結果を X（Twitter）・Instagram・共有で「SHARE YOUR SPRAY」
- **グローバルランキング**: シーズン制、トップ3・リスト・自分の順位バナーで競い合い
- **グループ対戦モード**（追加予定）: 人数を入力してルーム作成 → 参加者が揃うまで待機 → 各自がプレイしてスコア送信 → グループ内の順位（1位〜）を表示。友達同士で対戦する用途

## 技術スタック

**クライアント: Swift（ネイティブ iOS）**

| 用途 | 技術 |
|------|------|
| UI | SwiftUI または UIKit |
| 加速度・振り検知 | Core Motion（CMMotionManager） |
| アニメーション | Core Animation / Metal（液体・パーティクル） |
| 音声 | AVFoundation（効果音・BGM） |
| 非同期・状態 | Combine / async-await |
| バックエンド連携 | Firebase iOS SDK（Auth, Firestore, Realtime DB, Analytics） |
| 多言語 | Localizable.strings（NSLocalizedString / String(localized:)） |

**バックエンド**: Firebase（Firestore, Cloud Functions, Authentication, Cloud Storage）

- **データ設計**: エンティティ・Enum・Firestore コレクション・リレーションは **[docs/data-design.json](docs/data-design.json)** に定義。

※ Android 版は将来 **Kotlin（ネイティブ）** で別プロジェクトとして検討可能。
